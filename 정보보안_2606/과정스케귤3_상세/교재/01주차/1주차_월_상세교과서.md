# 1주차 월요일: Python 첫 실행과 입력 출력

> 오늘 만들 것: 자기소개 출력 프로그램, 입력형 자기소개 프로그램, 간단 계산기 초안  
> 오늘의 핵심 문장: 프로그램은 **입력 -> 처리 -> 출력** 순서로 움직인다.

## 0. 오늘 한 장 그림

```text
사람이 입력한다              컴퓨터가 기억하고 처리한다              화면에 보여 준다

이름: 민수        ----->      name = "민수"                  ----->    안녕하세요, 민수님
나이: 20                    age = "20"                              내년에는 21살입니다
```

아직 어려운 문법은 필요 없다. 오늘은 아래 세 가지를 확실히 잡는다.

| 할 일 | Python 코드 | 쉬운 설명 |
|---|---|---|
| 화면에 보여 주기 | `print()` | 괄호 안의 내용을 화면에 출력한다. |
| 키보드 입력 받기 | `input()` | 사용자가 입력한 글자를 받아 온다. |
| 값 기억하기 | `name = "민수"` | 오른쪽 값을 왼쪽 이름에 저장한다. |

## 1. 가장 작은 Python 프로그램

먼저 `hello.py` 파일을 만들고 아래 코드를 입력한다.

```python
print("안녕하세요")
```

실행 결과:

```text
안녕하세요
```

### 코드 읽기

```text
print ( "안녕하세요" )
출력해라   이 문장을
```

`print`는 "화면에 보여 줘"라는 뜻으로 생각하면 된다. 따옴표 안에 있는 글자는 그대로 출력된다.

## 2. 여러 줄 출력하기

```python
print("이름: 김민수")
print("관심 분야: 보안")
print("오늘 목표: Python 실행해 보기")
```

실행 결과:

```text
이름: 김민수
관심 분야: 보안
오늘 목표: Python 실행해 보기
```

### 눈으로 보는 실행 순서

```text
1번째 줄 실행 -> 이름 출력
2번째 줄 실행 -> 관심 분야 출력
3번째 줄 실행 -> 오늘 목표 출력
```

Python은 특별한 일이 없으면 위에서 아래로 한 줄씩 실행한다.

## 3. 변수는 이름표가 붙은 상자다

같은 이름을 여러 번 쓰고 싶다면 변수를 사용한다.

```python
name = "김민수"

print(name)
print(name)
print(name)
```

### 변수 상자 그림

```text
┌──────────────┐
│ name         │
│ "김민수"      │
└──────────────┘
```

`name = "김민수"`는 수학에서 말하는 "같다"가 아니다. Python에서는 이렇게 읽는 편이 좋다.

```text
오른쪽 값 "김민수"를 왼쪽 이름 name에 저장한다.
```

## 4. 입력 받기

이번에는 사용자가 직접 이름을 입력하게 한다.

```python
name = input("이름을 입력하세요: ")

print("안녕하세요")
print(name)
```

실행 예:

```text
이름을 입력하세요: 지연
안녕하세요
지연
```

### 입력 흐름

```text
input("이름을 입력하세요: ")
        |
        v
사용자가 "지연" 입력
        |
        v
name 상자에 "지연" 저장
        |
        v
print(name)에서 "지연" 출력
```

## 5. 문장 안에 변수 넣기

아래 코드는 이름을 더 자연스럽게 출력한다.

```python
name = input("이름을 입력하세요: ")

print("안녕하세요, " + name + "님")
```

실행 예:

```text
이름을 입력하세요: 지연
안녕하세요, 지연님
```

여기서 `+`는 숫자 더하기가 아니라 글자 이어 붙이기다.

```text
"안녕하세요, " + "지연" + "님"
=> "안녕하세요, 지연님"
```

## 6. 숫자 입력의 함정

아래 코드를 실행해 보자.

```python
first = input("첫 번째 숫자: ")
second = input("두 번째 숫자: ")

result = first + second

print(result)
```

실행 예:

```text
첫 번째 숫자: 10
두 번째 숫자: 20
1020
```

왜 `30`이 아니라 `1020`일까?

`input()`은 사용자가 숫자처럼 입력해도 **글자**로 받아 온다.

```text
사용자 입력      Python이 받은 값
10          ->  "10"
20          ->  "20"

"10" + "20" -> "1020"
```

## 7. `int()`로 숫자로 바꾸기

숫자 계산을 하려면 `int()`를 사용한다.

```python
first = input("첫 번째 숫자: ")
second = input("두 번째 숫자: ")

first_number = int(first)
second_number = int(second)

result = first_number + second_number

print(result)
```

실행 예:

```text
첫 번째 숫자: 10
두 번째 숫자: 20
30
```

### 형변환 그림

```text
"10"             int("10")             10
글자 숫자   ----------------->   진짜 숫자
```

## 8. 조금 더 짧게 쓰기

아래처럼 입력받자마자 숫자로 바꿀 수도 있다.

```python
first_number = int(input("첫 번째 숫자: "))
second_number = int(input("두 번째 숫자: "))

result = first_number + second_number

print(result)
```

처음에는 길게 써도 된다. 중요한 것은 "입력은 글자이고, 계산하려면 숫자로 바꾼다"는 점이다.

## 9. 손으로 추적하기

아래 코드를 실행하기 전에 표를 먼저 채워 보자.

```python
name = "민수"
age = 20
next_age = age + 1

print(name)
print(next_age)
```

| 실행 줄 | `name` | `age` | `next_age` | 출력 |
|---|---|---:|---:|---|
| `name = "민수"` | 민수 | - | - | - |
| `age = 20` | 민수 | 20 | - | - |
| `next_age = age + 1` | 민수 | 20 | 21 | - |
| `print(name)` | 민수 | 20 | 21 | 민수 |
| `print(next_age)` | 민수 | 20 | 21 | 21 |

이 표를 쓰는 습관이 정보처리기사 코드 추적의 시작이다.
