# 1주차 화요일: 자료형과 조건문

> 오늘 만들 것: 합격/불합격 판정기, 등급 판정기, 메뉴 선택 프로그램  
> 오늘의 핵심 문장: 조건문은 프로그램을 여러 갈래 길로 나누는 도구다.

## 0. 오늘 한 장 그림

```text
점수 입력
   |
   v
점수가 60 이상인가?
   |
   +-- 예  -> 합격 출력
   |
   +-- 아니오 -> 불합격 출력
```

조건문은 "상황에 따라 다른 일을 하게 하는 문법"이다.

## 1. 자료형 다시 보기

Python은 값의 종류를 구분한다.

| 값 | 자료형 | 의미 |
|---|---|---|
| `"안녕"` | 문자열 | 글자 |
| `"10"` | 문자열 | 숫자처럼 보이지만 글자 |
| `10` | 정수 | 계산 가능한 숫자 |
| `3.14` | 실수 | 소수점이 있는 숫자 |

확인하려면 `type()`을 사용한다.

```python
print(type("10"))
print(type(10))
```

실행 결과:

```text
<class 'str'>
<class 'int'>
```

`str`은 문자열, `int`는 정수다.

## 2. 비교 연산자

조건문은 보통 비교에서 시작한다.

| 연산자 | 뜻 | 예시 |
|---|---|---|
| `>` | 크다 | `score > 60` |
| `>=` | 크거나 같다 | `score >= 60` |
| `<` | 작다 | `score < 60` |
| `<=` | 작거나 같다 | `score <= 100` |
| `==` | 같다 | `menu == 1` |
| `!=` | 같지 않다 | `menu != 0` |

비교 결과는 `True` 또는 `False`다.

```python
score = 70

print(score >= 60)
print(score < 60)
```

실행 결과:

```text
True
False
```

## 3. `if` 기본 구조

```python
score = int(input("점수: "))

if score >= 60:
    print("합격입니다.")
```

### 들여쓰기 그림

```text
if score >= 60:
    print("합격입니다.")
^^^^
이 안쪽으로 들어간 줄은 조건이 참일 때만 실행된다.
```

Python에서는 들여쓰기가 아주 중요하다. 들여쓰기가 틀리면 코드의 의미가 달라지거나 에러가 난다.

## 4. `if / else`

합격이 아닌 경우도 처리하려면 `else`를 쓴다.

```python
score = int(input("점수: "))

if score >= 60:
    print("합격입니다.")
else:
    print("불합격입니다.")
```

### 실행 흐름

```text
score = 75
   |
score >= 60 ? True
   |
합격입니다 출력

score = 40
   |
score >= 60 ? False
   |
불합격입니다 출력
```

## 5. 손으로 추적하기

아래 코드에서 입력이 `55`라면 무엇이 출력될까?

```python
score = 55

if score >= 60:
    print("합격")
else:
    print("불합격")
```

| 변수 | 값 |
|---|---:|
| `score` | 55 |
| `score >= 60` | False |
| 실행되는 줄 | `else` 안쪽 |
| 출력 | 불합격 |

## 6. `elif`로 여러 갈래 만들기

등급처럼 여러 구간이 있으면 `elif`를 사용한다.

```python
score = int(input("점수: "))

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("보충")
```

### 왜 순서가 중요할까?

`score`가 85라고 해 보자.

```text
score >= 90 ? False
score >= 80 ? True  -> B 출력 후 끝
score >= 70 ? 검사하지 않음
```

조건문은 위에서 아래로 내려오다가 처음 맞는 곳을 실행한다.

## 7. 잘못된 순서 예시

```python
score = 95

if score >= 70:
    print("C")
elif score >= 80:
    print("B")
elif score >= 90:
    print("A")
```

95점인데 `C`가 출력된다. 첫 번째 조건 `score >= 70`이 이미 참이기 때문이다.

## 8. 복합 조건 `and`, `or`

조건을 두 개 이상 묶을 때 사용한다.

| 코드 | 뜻 |
|---|---|
| `a and b` | a도 참이고 b도 참이면 참 |
| `a or b` | a나 b 중 하나라도 참이면 참 |

예시:

```python
score = int(input("점수: "))
attendance = int(input("출석률: "))

if score >= 60 and attendance >= 80:
    print("최종 합격")
else:
    print("최종 불합격")
```

### 조건 표

| 점수 조건 | 출석 조건 | 전체 결과 |
|---|---|---|
| True | True | True |
| True | False | False |
| False | True | False |
| False | False | False |

`and`는 둘 다 통과해야 한다.

## 9. 메뉴 선택 프로그램

금요일 숫자 게임에서는 난이도를 선택해야 한다. 오늘은 메뉴 선택의 기본을 연습한다.

```python
print("1. 인사")
print("2. 계산")
print("3. 종료")

menu = int(input("메뉴 번호: "))

if menu == 1:
    print("안녕하세요.")
elif menu == 2:
    print("계산 기능을 선택했습니다.")
elif menu == 3:
    print("프로그램을 종료합니다.")
else:
    print("없는 메뉴입니다.")
```

## 12. 오늘 꼭 기억할 것

- 조건문의 결과는 참 또는 거짓이다.
- `if` 안쪽 코드는 조건이 참일 때 실행된다.
- `else`는 위 조건이 거짓일 때 실행된다.
- `elif`는 여러 갈래 중 하나를 고를 때 쓴다.
- 조건문의 순서는 결과에 영향을 준다.
