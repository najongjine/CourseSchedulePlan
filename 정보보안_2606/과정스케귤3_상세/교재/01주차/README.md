# 1주차 상세 교과서: Python 첫 감각과 콘솔 숫자 게임

## 이번 주의 목표

이번 주는 Python 문법을 많이 외우는 주가 아니다. 비전공자 수강생이 "코드는 위에서 아래로 실행되고, 입력값에 따라 결과가 달라진다"는 감각을 몸으로 익히는 주다.

최종 결과물은 **콘솔 숫자 맞히기 게임**이다. 월요일에는 출력과 입력, 화요일에는 조건문, 수요일에는 반복문, 목요일에는 랜덤과 힌트, 금요일에는 전체 게임 완성을 다룬다.

## 이번 주 전체 그림

```text
월요일              화요일              수요일              목요일              금요일
출력/입력/변수  ->  조건문 판정    ->  반복과 누적    ->  랜덤과 힌트    ->  숫자 게임 완성

print/input         if/elif/else        while/count         random             project
```

## 수업 방식

각 교재는 같은 흐름으로 구성되어 있다.

```text
눈으로 보기 -> 기본 문법 -> 따라 치는 예제 -> 손으로 추적 -> 연습문제 -> 미니 과제
```

처음부터 완성 코드를 외우지 않는다. 작은 코드를 실행하고, 값을 바꾸고, 에러를 고치면서 감각을 만든다.
