# 1주차 목요일: random과 업다운 힌트

> 오늘 만들 것: 랜덤 운세 뽑기, 업다운 숫자 게임 1차 버전, 난이도 선택 초안  
> 오늘의 핵심 문장: 랜덤 정답, 조건문, 반복문을 합치면 게임 흐름이 된다.

## 0. 오늘 한 장 그림

```text
랜덤 정답 만들기
   |
사용자 숫자 입력
   |
정답과 비교
   |
   +-- 입력 < 정답 -> 더 큰 수 안내
   +-- 입력 > 정답 -> 더 작은 수 안내
   +-- 입력 == 정답 -> 성공 후 종료
```

오늘은 월, 화, 수에 배운 내용을 합친다.

| 배운 내용 | 오늘 쓰는 곳 |
|---|---|
| `input()` | 사용자가 숫자를 입력한다. |
| `int()` | 입력값을 숫자로 바꾼다. |
| `if` | 정답보다 큰지 작은지 비교한다. |
| `while` | 여러 번 시도한다. |

## 1. 랜덤이 필요한 이유

숫자 맞히기 게임에서 정답이 항상 7이면 금방 재미가 없어진다. 실행할 때마다 정답이 달라져야 게임이 된다.

Python에서는 `random`을 사용한다.

```python
import random

number = random.randint(1, 10)

print(number)
```

실행할 때마다 1부터 10 사이 숫자 하나가 나온다.

## 2. `import random` 읽기

```text
import random
랜덤 기능을 가져와서 사용할 준비를 한다.
```

`random.randint(1, 10)`은 이렇게 읽는다.

```text
1부터 10까지 중에서 정수 하나를 뽑아라.
```

여기서 1과 10도 포함된다.

## 3. 랜덤 운세 뽑기

숫자 게임 전에 간단한 랜덤 프로그램으로 감을 잡자.

```python
import random

number = random.randint(1, 3)

if number == 1:
    print("오늘은 차분히 가면 좋은 날입니다.")
elif number == 2:
    print("작은 실습을 끝까지 해내는 날입니다.")
else:
    print("에러를 고치며 실력이 느는 날입니다.")
```

### 실행 흐름

```text
random.randint(1, 3)
        |
        v
1, 2, 3 중 하나
        |
        v
조건문으로 문장 선택
```

## 4. 정답 만들고 입력받기

```python
import random

answer = random.randint(1, 10)
guess = int(input("1부터 10 사이 숫자 입력: "))

print("정답:", answer)
print("입력:", guess)
```

처음 개발할 때는 정답을 출력해도 된다. 그래야 코드가 맞게 움직이는지 확인할 수 있다. 하지만 게임 완성본에서는 정답을 숨겨야 한다.

## 5. 업다운 힌트 만들기

```python
import random

answer = random.randint(1, 10)
guess = int(input("1부터 10 사이 숫자 입력: "))

if guess < answer:
    print("더 큰 수입니다.")
elif guess > answer:
    print("더 작은 수입니다.")
else:
    print("정답입니다.")
```

### 비교 그림

정답이 7이라고 하자.

```text
입력 3  ->  3 < 7  -> 더 큰 수입니다.
입력 9  ->  9 > 7  -> 더 작은 수입니다.
입력 7  ->  7 == 7 -> 정답입니다.
```

## 6. 테스트할 때는 정답을 고정하기

랜덤은 매번 바뀌어서 테스트가 어렵다. 개발 중에는 잠깐 이렇게 고정해도 된다.

```python
answer = 7
```

테스트가 끝나면 다시 랜덤으로 바꾼다.

```python
answer = random.randint(1, 10)
```

## 7. 반복해서 묻기

사용자가 한 번에 못 맞힐 수 있으므로 반복문을 붙인다.

```python
import random

answer = random.randint(1, 10)
try_count = 1

while try_count <= 5:
    guess = int(input("숫자 입력: "))

    if guess < answer:
        print("더 큰 수입니다.")
    elif guess > answer:
        print("더 작은 수입니다.")
    else:
        print("정답입니다.")
        break

    try_count = try_count + 1
```

## 8. 남은 기회 보여 주기

사용자가 지금 몇 번째 시도인지 알면 더 게임답다.

```python
import random

answer = random.randint(1, 10)
try_count = 1
max_try = 5

while try_count <= max_try:
    print("현재 시도:", try_count, "/", max_try)
    guess = int(input("숫자 입력: "))

    if guess < answer:
        print("더 큰 수입니다.")
    elif guess > answer:
        print("더 작은 수입니다.")
    else:
        print("정답입니다.")
        break

    try_count = try_count + 1
```

## 9. 난이도 선택 초안

금요일에는 난이도를 붙일 것이다. 오늘은 변수만 바꿔 보는 구조를 만든다.

```python
print("1. 쉬움")
print("2. 보통")
print("3. 어려움")

level = int(input("난이도 선택: "))

if level == 1:
    max_number = 10
    max_try = 5
elif level == 2:
    max_number = 50
    max_try = 7
else:
    max_number = 100
    max_try = 10

print("숫자 범위: 1부터", max_number)
print("시도 횟수:", max_try)
```

### 난이도 변수 그림

```text
쉬움 선택  -> max_number = 10,  max_try = 5
보통 선택  -> max_number = 50,  max_try = 7
어려움 선택 -> max_number = 100, max_try = 10
```

정답을 만들 때 `max_number`를 사용하면 된다.

```python
answer = random.randint(1, max_number)
```

## 12. 오늘 꼭 기억할 것

- `import random`은 랜덤 기능을 가져온다.
- `random.randint(1, 10)`은 1부터 10 사이 정수 하나를 만든다.
- 게임은 정답 만들기, 입력받기, 비교하기, 반복하기로 나눌 수 있다.
