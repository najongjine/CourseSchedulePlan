## 1. 오늘 완성할 게임 그림


## 3. 1단계: 시작 메시지

```python
print("숫자 맞히기 게임")
print("컴퓨터가 고른 숫자를 맞혀 보세요.")
```

작지만 중요한 시작이다. 사용자에게 지금 어떤 프로그램인지 알려 준다.

## 4. 2단계: 난이도 선택

```python
print("난이도를 선택하세요.")
print("1. 쉬움")
print("2. 보통")
print("3. 어려움")

level = int(input("선택: "))

if level == 1:
    max_number = 10
    max_try = 5
elif level == 2:
    max_number = 50
    max_try = 7
else:
    max_number = 100
    max_try = 10

print("1부터", max_number, "사이의 숫자를 맞히세요.")
print("기회는", max_try, "번입니다.")
```

### 변수 그림

```text
level          max_number          max_try
  1       ->       10          ->      5
  2       ->       50          ->      7
  3       ->       100         ->      10
```

난이도 선택은 결국 게임에 사용할 숫자 두 개를 정하는 일이다.

## 5. 3단계: 정답 만들기

```python
import random

answer = random.randint(1, max_number)
```

위 코드는 `1`부터 `max_number` 사이에서 정답을 만든다.

개발 중에는 아래 코드를 잠깐 넣어 테스트할 수 있다.

```python
print("테스트용 정답:", answer)
```

최종 제출 전에는 지운다.

## 6. 4단계: 입력 한 번 받고 비교하기

```python
guess = int(input("숫자 입력: "))

if guess < answer:
    print("더 큰 수입니다.")
elif guess > answer:
    print("더 작은 수입니다.")
else:
    print("정답입니다.")
```

정답이 30일 때:

```text
입력 10 -> 더 큰 수입니다.
입력 40 -> 더 작은 수입니다.
입력 30 -> 정답입니다.
```

## 7. 5단계: 여러 번 시도하기

```python
try_count = 1

while try_count <= max_try:
    print("현재 시도:", try_count, "/", max_try)
    guess = int(input("숫자 입력: "))

    if guess < answer:
        print("더 큰 수입니다.")
    elif guess > answer:
        print("더 작은 수입니다.")
    else:
        print("정답입니다.")
        break

    try_count = try_count + 1
```

### 반복 변수 추적

`max_try`가 5라면:

| 반복 | `try_count <= max_try` | 입력 기회 | 반복 끝의 `try_count` |
|---:|---|---|---:|
| 1 | True | 1번째 기회 | 2 |
| 2 | True | 2번째 기회 | 3 |
| 3 | True | 3번째 기회 | 4 |
| 4 | True | 4번째 기회 | 5 |
| 5 | True | 5번째 기회 | 6 |
| 6 | False | 없음 | 6 |

## 8. 6단계: 성공/실패 구분하기

반복문이 끝났을 때 왜 끝났는지 알아야 한다.

- 정답을 맞혀서 끝났는가?
- 기회를 모두 써서 끝났는가?

이를 위해 `success` 변수를 사용한다.

```python
success = False
```

처음에는 아직 성공하지 않았으므로 `False`다. 정답을 맞히면 `True`로 바꾼다.

```python
if guess == answer:
    success = True
    print("정답입니다.")
    break
```

반복문이 끝난 뒤 확인한다.

```python
if success == False:
    print("실패했습니다.")
    print("정답은", answer, "입니다.")
```

## 9. 7단계: 기본 완성 코드

아래 코드는 재시작 기능을 제외한 기본 완성본이다.

```python
import random

print("숫자 맞히기 게임")
print("난이도를 선택하세요.")
print("1. 쉬움")
print("2. 보통")
print("3. 어려움")

level = int(input("선택: "))

if level == 1:
    max_number = 10
    max_try = 5
elif level == 2:
    max_number = 50
    max_try = 7
else:
    max_number = 100
    max_try = 10

answer = random.randint(1, max_number)
try_count = 1
success = False

print("1부터", max_number, "사이의 숫자를 맞히세요.")
print("기회는", max_try, "번입니다.")

while try_count <= max_try:
    print("현재 시도:", try_count, "/", max_try)
    guess = int(input("숫자 입력: "))

    if guess < answer:
        print("더 큰 수입니다.")
    elif guess > answer:
        print("더 작은 수입니다.")
    else:
        success = True
        print("정답입니다.")
        print(try_count, "번 만에 맞혔습니다.")
        break

    try_count = try_count + 1

if success == False:
    print("실패했습니다.")
    print("정답은", answer, "입니다.")
```

## 10. 8단계: 재시작 기능 붙이기

게임 전체를 `while True`로 감싸고, 마지막에 다시 할지 묻는다.

```python
while True:
    print("게임을 시작합니다.")

    again = input("다시 하시겠습니까? y/n: ")

    if again != "y":
        break
```

핵심 구조는 이렇다.

```text
계속 반복
   |
게임 한 판 실행
   |
다시 할지 묻기
   |
   +-- y가 아니면 break
   +-- y이면 다시 처음으로
```

## 11. 최종 코드

파일명: `number_guessing_game.py`

```python
import random

while True:
    print("숫자 맞히기 게임")
    print("난이도를 선택하세요.")
    print("1. 쉬움")
    print("2. 보통")
    print("3. 어려움")

    level = int(input("선택: "))

    if level == 1:
        max_number = 10
        max_try = 5
    elif level == 2:
        max_number = 50
        max_try = 7
    else:
        max_number = 100
        max_try = 10

    answer = random.randint(1, max_number)
    try_count = 1
    success = False

    print("1부터", max_number, "사이의 숫자를 맞히세요.")
    print("기회는", max_try, "번입니다.")

    while try_count <= max_try:
        print("현재 시도:", try_count, "/", max_try)
        guess = int(input("숫자 입력: "))

        if guess < answer:
            print("더 큰 수입니다.")
        elif guess > answer:
            print("더 작은 수입니다.")
        else:
            success = True
            print("정답입니다.")
            print(try_count, "번 만에 맞혔습니다.")
            break

        try_count = try_count + 1

    if success == False:
        print("실패했습니다.")
        print("정답은", answer, "입니다.")

    again = input("다시 하시겠습니까? y/n: ")

    if again != "y":
        print("게임을 종료합니다.")
        break
```

## 12. 코드 해독하기

| 변수 | 역할 |
|---|---|
| `level` | 사용자가 고른 난이도 |
| `max_number` | 정답 숫자의 최대 범위 |
| `max_try` | 최대 시도 횟수 |
| `answer` | 컴퓨터가 만든 정답 |
| `try_count` | 현재 몇 번째 시도인지 세는 변수 |
| `guess` | 사용자가 입력한 숫자 |
| `success` | 정답을 맞혔는지 기억하는 변수 |
| `again` | 다시 할지 묻는 입력값 |

## 13. 테스트하기

랜덤 때문에 테스트가 어렵다면 잠깐 정답을 고정한다.

```python
answer = 7
```

그 다음 아래 표대로 테스트한다.

| 테스트 | 입력 | 기대 결과 |
|---|---|---|
| 작은 수 입력 | 3 | 더 큰 수입니다. |
| 큰 수 입력 | 9 | 더 작은 수입니다. |
| 정답 입력 | 7 | 정답입니다. |
| 모두 실패 | 1, 2, 3, 4, 5 | 실패 메시지와 정답 출력 |
| 재시작 안 함 | n | 게임 종료 |

테스트가 끝나면 다시 랜덤으로 돌려놓는다.

```python
answer = random.randint(1, max_number)
```
