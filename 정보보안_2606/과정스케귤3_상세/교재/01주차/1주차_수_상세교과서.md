# 1주차 수요일: while 반복문과 누적

> 오늘 만들 것: 반복 출력 프로그램, 시도 횟수 제한 퀴즈, 점수 누적 프로그램  
> 오늘의 핵심 문장: 반복문은 같은 일을 조건이 맞는 동안 계속 실행한다.

## 0. 오늘 한 장 그림

```text
count = 1
   |
count <= 3 인가?
   |
   +-- 예  -> 실행 -> count 1 증가 -> 다시 조건 검사
   |
   +-- 아니오 -> 반복 종료
```

반복문에서 가장 중요한 질문은 두 가지다.

1. 언제 반복할 것인가?
2. 언제 멈출 것인가?

## 1. 반복 없이 세 번 출력하기

```python
print("안녕하세요")
print("안녕하세요")
print("안녕하세요")
```

세 번이면 괜찮다. 하지만 100번이면 같은 코드를 100줄 써야 한다. 이럴 때 반복문을 쓴다.

## 2. `while` 기본 구조

```python
count = 1

while count <= 3:
    print("안녕하세요")
    count = count + 1
```

실행 결과:

```text
안녕하세요
안녕하세요
안녕하세요
```

### 코드 해석

| 코드 | 뜻 |
|---|---|
| `count = 1` | 반복 횟수를 셀 변수를 1로 시작 |
| `while count <= 3:` | count가 3 이하인 동안 반복 |
| `print("안녕하세요")` | 반복할 일 |
| `count = count + 1` | count를 1 증가 |

## 3. 반복 추적표

```text
처음 count = 1
```

| 반복 차례 | 조건 `count <= 3` | 출력 | 반복 끝의 `count` |
|---:|---|---|---:|
| 1 | True | 안녕하세요 | 2 |
| 2 | True | 안녕하세요 | 3 |
| 3 | True | 안녕하세요 | 4 |
| 4 | False | 출력 없음 | 4 |

`count`가 4가 되면 조건이 거짓이 되어 반복이 끝난다.

## 4. 무한 반복 조심하기

아래 코드는 끝나지 않는다.

```python
count = 1

while count <= 3:
    print("안녕하세요")
```

왜일까?

`count`가 계속 1이기 때문이다. `count = count + 1`이 없으면 조건이 영원히 참이다.

```text
count = 1
count <= 3 ? True
출력
count 여전히 1
count <= 3 ? True
출력
...
```

## 5. 시도 횟수 제한 퀴즈

정답을 맞히는 퀴즈를 만들어 보자. 아직 랜덤은 쓰지 않는다.

```python
answer = 7
try_count = 1

while try_count <= 3:
    guess = int(input("숫자 입력: "))

    if guess == answer:
        print("정답입니다.")
        break
    else:
        print("틀렸습니다.")

    try_count = try_count + 1
```

### 새로 나온 `break`

`break`는 반복문을 바로 끝내는 명령이다.

```text
반복 중 정답을 맞힘
   |
   v
break 실행
   |
   v
while 밖으로 나감
```

## 6. 누적 계산

점수 세 개를 입력받아 합계를 구해 보자.

```python
total = 0
count = 1

while count <= 3:
    score = int(input("점수: "))
    total = total + score
    count = count + 1

print("합계:", total)
```

### 누적 그림

입력값이 80, 90, 70이라면:

```text
처음 total = 0

1번째 입력 80 -> total = 0 + 80  -> 80
2번째 입력 90 -> total = 80 + 90 -> 170
3번째 입력 70 -> total = 170 + 70 -> 240
```

## 7. 평균까지 계산하기

```python
total = 0
count = 1

while count <= 3:
    score = int(input("점수: "))
    total = total + score
    count = count + 1

average = total / 3

print("합계:", total)
print("평균:", average)
```

`/`는 나눗셈이다. 평균은 합계를 개수로 나눈 값이다.

## 8. 0을 입력할 때까지 반복하기

반복 횟수를 미리 모를 때도 있다. 예를 들어 사용자가 0을 입력하면 종료하게 만들 수 있다.

```python
total = 0

while True:
    number = int(input("숫자 입력, 종료는 0: "))

    if number == 0:
        break

    total = total + number

print("합계:", total)
```

### `while True` 그림

```text
계속 반복
   |
숫자 입력
   |
0인가?
   +-- 예 -> break로 종료
   +-- 아니오 -> total에 더하고 계속
```

## 10. 오늘의 미니 과제

파일명: `try_limit_quiz.py`

정답 숫자를 `5`로 정해 둔다. 사용자는 최대 3번까지 숫자를 입력할 수 있다.

조건:

- 맞히면 `정답입니다.` 출력 후 종료
- 틀리면 `틀렸습니다.` 출력
- 3번 모두 틀리면 `기회를 모두 사용했습니다.` 출력

힌트:

```python
success = False
```

성공 여부를 기억하는 변수를 사용하면 마지막에 성공/실패를 구분하기 쉽다.

## 11. 성공 변수 예시

```python
answer = 5
try_count = 1
success = False

while try_count <= 3:
    guess = int(input("숫자 입력: "))

    if guess == answer:
        success = True
        print("정답입니다.")
        break
    else:
        print("틀렸습니다.")

    try_count = try_count + 1

if success == False:
    print("기회를 모두 사용했습니다.")
```

## 12. 오늘 꼭 기억할 것

- `while`은 조건이 참인 동안 반복한다.
- 반복문에는 멈추는 조건이 반드시 필요하다.
- `count = count + 1`은 횟수를 세는 기본 패턴이다.
- `total = total + score`는 누적 계산의 기본 패턴이다.
- `break`는 반복문을 바로 끝낸다.
