- `while` 반복문의 기본 구조를 익힌다.

## 수업 전개 뼈대

### 1. 반복 없이 세 번 출력하기

- 같은 `print()`를 세 번 작성해 불편함을 먼저 느낀다.
- `while`로 같은 결과를 만드는 코드를 작성한다.
- 반복문은 "복사 붙여넣기 줄이기"에서 출발한다고 설명한다.

### 2. 카운터로 횟수 세기

- `count = 1`에서 시작해 `count = count + 1`로 증가시킨다.
- 반복 조건을 `count <= 3`으로 두고 3번만 실행한다.
- 정보처리기사식 변수 추적표로 `count` 변화를 적는다.

### 3. 시도 횟수 제한 퀴즈

- 정답 숫자를 하나 정해 두고 사용자가 세 번 안에 맞히게 한다.
- 맞히면 성공, 못 맞히면 실패 메시지를 출력한다.
- 아직 랜덤은 쓰지 않고, 반복 구조와 조건 결합에 집중한다.

### 4. 누적 점수 프로그램

- 점수를 3개 입력받아 합계와 평균을 계산한다.
- `total = total + score`가 왜 누적인지 그림으로 설명한다.
- 금요일 게임의 시도 횟수, 점수, 기록으로 연결한다.
