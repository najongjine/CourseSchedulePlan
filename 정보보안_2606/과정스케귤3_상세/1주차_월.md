## 수업 전개 뼈대

### 1. 가장 작은 프로그램 실행

- `hello.py`를 만들고 `print("안녕하세요")`를 실행한다.
- 문장을 바꿔 보며 코드 수정 후 다시 실행하는 흐름을 반복한다.
- 에러가 나면 메시지 전체를 무서워하지 않고, 어느 줄에서 났는지 먼저 본다.

### 2. 자기소개 출력 프로그램

- 이름, 관심사, 오늘 목표를 고정 문장으로 출력한다.
- 여러 줄 출력과 문자열 연결을 연습한다.
- 같은 값을 여러 번 쓰고 싶을 때 변수가 필요하다는 흐름으로 넘어간다.

### 3. 입력받는 자기소개 프로그램

- `name = input("이름: ")` 형태로 입력값을 받는다.
- 입력값을 다시 문장에 끼워 넣어 출력한다.
- 변수 이름을 바꾸면 뒤쪽 코드도 같이 바뀌어야 한다는 점을 확인한다.

### 4. 간단 계산기 초안

- 두 숫자를 입력받는다.
- 처음에는 문자열끼리 더해지는 문제를 일부러 경험한다.
- `int()`로 바꾼 뒤 합계, 차이, 곱셈 결과를 출력한다.
