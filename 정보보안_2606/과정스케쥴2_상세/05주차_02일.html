반복해서 다섯 날 기록하기

월요일부터 금요일까지의 공부시간을 반복 입력하고 총 시간을 누적한다.

5주차 2일차오늘의 목표4시간 흐름감각으로 이해하기직접 따라치기기사형 코드 추적손으로 푸는 연습금요일 프로젝트 연결
이전다음

오늘의 목표

  • `for`와 `range`를 읽는다.
  • 누적 합계 변화를 기록한다.
  • 반복 횟수를 추적한다.

오늘 4시간의 흐름

1교시개념을 그림과 코드로 이해
2교시예제를 직접 입력하고 실행
3교시금요일 작품 기능 한 조각 작성
4교시값을 바꾸고 손 추적 연습

먼저 감각으로 이해하기

반복문은 다섯 칸짜리 기록표를 한 칸씩 채우는 손

`range(5)`는 0,1,2,3,4라는 다섯 값을 차례로 제공한다.

`total`은 매일 입력이 들어올 때마다 이전 값에 새 시간을 더해 기억한다.

day=0+2→day=1+3→day=2+1→total6 ...

손가락 실습: 월~금 다섯 칸을 그리고 입력을 들을 때마다 total 옆의 숫자를 덧셈 결과로 바꾼다.

직접 따라치기

우선 리스트 없이 합계만 계산해 반복과 누적에 집중한다.

코드를 입력한 뒤 반드시 실행한다

total = 0

for day in range(5):
    hours = int(input(f"{day + 1}일차 공부시간: "))
    total = total + hours
    print("지금까지:", total, "시간")

print("주간 총 공부시간:", total, "시간")

실행 결과 예

1일차 공부시간: 2 지금까지: 2 시간 2일차 공부시간: 3 지금까지: 5 시간 ... 주간 총 공부시간: 10 시간

한 줄씩 읽어 보기

  1. `day`는 화면 번호를 위해 1을 더해 출력하지만 반복 횟수는 다섯 번이다.
  2. `total`은 반복 밖에서 만들어져 이전 날의 합계를 유지한다.
  3. 마지막 print는 반복이 끝난 후 한 번만 실행된다.
입력한 뒤 바꾸어 볼 값

입력을 모두 1로 넣을 때 각 회차의 total을 종이에 먼저 적는다.

손으로 추적하는 문제

누적 문제에서는 초기값 0과 반복 횟수를 먼저 찾는다.

total = 1
for n in range(1, 4):
    total = total * n
print(total)

문제. `n`이 어떤 값을 거치고 무엇이 출력되는가?

정답과 풀이 열기

`range(1,4)`는 1,2,3이다. total은 1, 2, 6이 되어 `6`을 출력한다.

감각을 굳히는 연습

먼저 종이에 예상 결과를 적고, 그 다음 코드를 바꾸어 실행해서 확인한다. 틀렸다면 틀린 이유가 학습 기록이다.

  1. 세 날만 입력받도록 range를 변경해 실행한다.
  2. 합계 대신 공부시간에 10을 곱한 점수를 누적해 본다.
  3. 반복 표에 day, hours, total 세 열을 채운다.
연습 확인 포인트

반복을 읽는 감각은 C의 `for`와 동일하며 Python 표현만 간결하다.
