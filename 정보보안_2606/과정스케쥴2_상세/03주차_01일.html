Java로 다시 만나는 입력과 출력

새 언어지만 값 저장과 입력/출력의 원리는 C와 같다. 문법 모양의 차이를 먼저 비교한다.

3주차 1일차오늘의 목표4시간 흐름감각으로 이해하기직접 따라치기기사형 코드 추적손으로 푸는 연습금요일 프로젝트 연결

오늘의 목표

  • Java 프로그램의 시작점을 찾는다.
  • `Scanner`로 정수를 입력받는다.
  • C와 Java의 같은 역할 코드를 연결한다.

오늘 4시간의 흐름

1교시개념을 그림과 코드로 이해
2교시예제를 직접 입력하고 실행
3교시금요일 작품 기능 한 조각 작성
4교시값을 바꾸고 손 추적 연습

먼저 감각으로 이해하기

언어가 바뀌어도 입력-처리-출력 뼈대는 같다

C의 `printf`, `scanf`와 Java의 `System.out.println`, `Scanner`는 생김새가 다르지만 사용자와 값을 주고받는 역할은 같다.

Java는 코드를 `class`라는 큰 상자 안에 적고 `main`에서 실행을 시작한다.

키보드10000→ScannernextInt()→balance10000→화면잔액 출력

손가락 실습: C에서 쓰던 입력/출력 문장과 Java 문장을 좌우로 적고 서로 선으로 연결한다.

직접 따라치기

`WalletApp.java` 파일로 저장하고 클래스 이름과 파일 이름을 같게 맞춘다.

코드를 입력한 뒤 반드시 실행한다

import java.util.Scanner;

public class WalletApp {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int balance;

        System.out.print("시작 용돈: ");
        balance = scanner.nextInt();
        System.out.println("현재 잔액: " + balance + "원");
    }
}

실행 결과 예

시작 용돈: 10000 현재 잔액: 10000원

한 줄씩 읽어 보기

  1. `new Scanner(System.in)`은 키보드 입력을 읽을 도구를 만든다.
  2. `nextInt()`가 읽은 정수는 `balance` 변수에 저장된다.
  3. 문자열 뒤의 `+ balance`는 출력 문장에 숫자를 이어 붙인다.
입력한 뒤 바꾸어 볼 값

시작 용돈을 5000으로 입력하고, 잔액 뒤에 `공부 파이팅` 문장을 추가해 출력한다.

손으로 추적하는 문제

Java 코드 문제도 대입 순서와 출력 조합을 묻는다. 새 문법 표면에 겁먹지 않고 값을 쫓는다.

int money = 1000;
money = money + 500;
money -= 200;
System.out.print(money);

문제. 최종 출력되는 잔액은 얼마인가?

정답과 풀이 열기

1000에서 500을 더해 1500, 200을 빼 1300이다. 출력은 `1300`.

감각을 굳히는 연습

먼저 종이에 예상 결과를 적고, 그 다음 코드를 바꾸어 실행해서 확인한다. 틀렸다면 틀린 이유가 학습 기록이다.

  1. 입금 금액 하나를 추가 입력받아 잔액과 합산 출력한다.
  2. C의 `int balance`와 Java의 `int balance`가 같은 점을 적는다.
  3. `+`가 숫자 더하기로 쓰일 때와 문자열 연결로 쓰일 때 예를 각각 만든다.
연습 확인 포인트

문자열이 먼저 나오면 숫자가 출력 문장에 이어지고, 두 정수끼리면 먼저 계산된다.
