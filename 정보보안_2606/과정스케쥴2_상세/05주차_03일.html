리스트에 기록을 남기기

합계만 남기지 않고 다섯 날의 값을 리스트에 보관하여 평균과 최고 기록을 구한다.

5주차 3일차오늘의 목표4시간 흐름감각으로 이해하기직접 따라치기기사형 코드 추적손으로 푸는 연습금요일 프로젝트 연결
이전다음

오늘의 목표

  • 리스트에 값을 추가한다.
  • 인덱스로 특정 날을 읽는다.
  • `sum`, `max`를 직접 계산과 비교한다.

오늘 4시간의 흐름

1교시개념을 그림과 코드로 이해
2교시예제를 직접 입력하고 실행
3교시금요일 작품 기능 한 조각 작성
4교시값을 바꾸고 손 추적 연습

먼저 감각으로 이해하기

리스트는 지우지 않는 주간 기록판

누적 total은 최종 합계만 남지만 리스트에는 매일의 값이 모두 남는다.

리스트 첫 칸도 배열처럼 0번이다. C 배열을 이미 본 학생에게는 같은 줄의 상자라고 연결한다.

hours[0]2→hours[1]4→hours[2]3→max4

손가락 실습: C 배열 그림 옆에 Python 리스트 그림을 나란히 그리고 인덱스가 같은 것을 표시한다.

직접 따라치기

입력을 리스트에 추가한 뒤 내장 함수로 결과를 확인한다.

코드를 입력한 뒤 반드시 실행한다

hours = []

for day in range(5):
    value = int(input(f"{day + 1}일차 시간: "))
    hours.append(value)

total = sum(hours)
average = total / len(hours)
print("기록:", hours)
print("총합:", total)
print("평균:", average)
print("최고:", max(hours))

실행 결과 예

1일차 시간: 2 2일차 시간: 4 3일차 시간: 3 4일차 시간: 1 5일차 시간: 5 기록: [2, 4, 3, 1, 5] 총합: 15 평균: 3.0 최고: 5

한 줄씩 읽어 보기

  1. `append`는 리스트 끝에 새 값을 한 칸 추가한다.
  2. `len(hours)`는 입력한 기록 개수 5다.
  3. `max`는 저장된 값 중 가장 큰 값 하나를 찾는다.
입력한 뒤 바꾸어 볼 값

`max`를 쓰지 않고 반복문과 `best` 변수로 최고 시간을 찾는 코드를 작성해 본다.

손으로 추적하는 문제

리스트 문제 역시 인덱스와 반복에 집중한다. 내장 함수 결과도 손으로 확인한다.

data = [3, 1, 4, 2]
result = 0
for i in range(0, len(data), 2):
    result += data[i]
print(result)

문제. 더해지는 리스트 칸과 출력값은 무엇인가?

정답과 풀이 열기

i는 0과 2이므로 data[0]=3과 data[2]=4를 더해 `7`이다.

감각을 굳히는 연습

먼저 종이에 예상 결과를 적고, 그 다음 코드를 바꾸어 실행해서 확인한다. 틀렸다면 틀린 이유가 학습 기록이다.

  1. 공부시간 `[2, 0, 4, 3, 1]`의 합계와 최고를 손으로 구한 뒤 코드로 확인한다.
  2. 최저시간을 출력하는 기능을 추가한다.
  3. 목표 3시간 이상인 값의 개수를 반복문으로 센다.
