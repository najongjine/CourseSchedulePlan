분석 규칙을 함수로 묶기

목표 달성일 수를 계산하는 기능을 함수로 만들고 주간 기록을 전달한다.

5주차 4일차오늘의 목표4시간 흐름감각으로 이해하기직접 따라치기기사형 코드 추적손으로 푸는 연습금요일 프로젝트 연결
이전다음

오늘의 목표

  • 함수의 매개변수로 리스트를 전달한다.
  • 조건에 맞는 개수를 누적한다.
  • 반환값이 저장되는 과정을 추적한다.

오늘 4시간의 흐름

1교시개념을 그림과 코드로 이해
2교시예제를 직접 입력하고 실행
3교시금요일 작품 기능 한 조각 작성
4교시값을 바꾸고 손 추적 연습

먼저 감각으로 이해하기

함수는 기록판을 받아 질문에 답하는 분석 담당자

기록 리스트를 함수에 주면 함수는 목표 이상인 날만 세고 숫자 하나를 돌려준다.

계산 규칙을 함수로 이름 붙이면 `count_success`라는 이름만 보고도 역할을 짐작할 수 있다.

hours[2,4,3,1,5]→함수>=3 세기→count3→출력달성일 3일

손가락 실습: 공부시간 카드 중 목표 이상인 카드만 한쪽으로 옮기고 몇 장인지 센다.

직접 따라치기

리스트는 예시값으로 먼저 실행한 뒤 사용자 입력과 결합한다.

코드를 입력한 뒤 반드시 실행한다

def count_success(hours, goal):
    count = 0
    for value in hours:
        if value >= goal:
            count += 1
    return count

records = [2, 4, 3, 1, 5]
goal = 3
success_days = count_success(records, goal)
print("목표 달성일:", success_days, "일")

실행 결과 예

목표 달성일: 3 일

한 줄씩 읽어 보기

  1. 함수 안 `value`는 리스트 값을 2,4,3,1,5 순서로 받는다.
  2. 4,3,5에서 조건이 참이므로 count가 세 번 증가한다.
  3. 반환된 3은 `success_days`에 저장된다.
입력한 뒤 바꾸어 볼 값

goal을 4로 변경할 때 참이 되는 값과 출력 숫자를 실행 전 적는다.

손으로 추적하는 문제

함수 문제에서는 반복 안의 조건이 몇 번 참인지 세면 반환값을 찾을 수 있다.

def change(items):
    result = 0
    for x in items:
        if x % 2 == 0:
            result += x
    return result
print(change([1, 2, 3, 4]))

문제. 출력은 무엇인가?

정답과 풀이 열기

짝수인 2와 4만 result에 더해져 `6`을 출력한다.

감각을 굳히는 연습

먼저 종이에 예상 결과를 적고, 그 다음 코드를 바꾸어 실행해서 확인한다. 틀렸다면 틀린 이유가 학습 기록이다.

  1. 목표 미달일 수를 반환하는 함수를 하나 더 만든다.
  2. 빈 리스트라면 평균 계산이 왜 문제가 되는지 설명한다.
  3. 내 기록 다섯 개를 넣고 함수 반환값을 표로 남긴다.
연습 확인 포인트
