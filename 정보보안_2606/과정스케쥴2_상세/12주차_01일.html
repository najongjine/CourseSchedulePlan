직접 따라치기

오늘 코드는 프로젝트 수정 대신 C 실기형 추적 복습 코드로 직접 입력한다.

코드를 입력한 뒤 반드시 실행한다

#include <stdio.h>

int main(void) {
    int stock[3] = {2, 0, 3};
    int available = 0;
    for (int i = 0; i < 3; i++) {
        if (stock[i] > 0) {
            available += stock[i];
        }
    }
    printf("%d", available);
    return 0;
}

실행 결과 예

5

한 줄씩 읽어 보기

  1. 배열의 재고를 하나씩 읽는다.
  2. 0보다 큰 2와 3만 available에 더한다.
  3. 이 로직은 화면의 판매 가능 재고 표시와도 연결할 수 있다.
입력한 뒤 바꾸어 볼 값

조건을 `>= 0`으로 바꾸어도 합계가 같은 이유와, 카운트를 셀 때는 달라질 수 있는 이유를 적는다.

손으로 추적하는 문제

기사 대비는 최종 작품 주에도 매일 한 언어씩 짧게 되짚어 감각을 유지한다.

int data[4] = {1, 2, 3, 4};
int result = 0;
for (int i = 0; i < 4; i++) {
    if (i % 2 == 0) result += data[i];
}
printf("%d", result);

문제. 출력값은 무엇인가?

정답과 풀이 열기

짝수 인덱스 0과 2의 값 1과 3을 더하므로 `4`이다.
