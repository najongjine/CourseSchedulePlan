짧게 쓰는 입력과 계산

Python에서는 중괄호와 자료형 선언이 줄어든다. 그러나 입력값을 숫자로 바꾸고 계산하는 원리는 그대로다.

5주차 1일차오늘의 목표4시간 흐름감각으로 이해하기직접 따라치기기사형 코드 추적손으로 푸는 연습금요일 프로젝트 연결

오늘의 목표

  • Python 파일을 실행한다.
  • `input()`과 `int()`의 역할을 구분한다.
  • C/Java/Python의 같은 흐름을 비교한다.

오늘 4시간의 흐름

1교시개념을 그림과 코드로 이해
2교시예제를 직접 입력하고 실행
3교시금요일 작품 기능 한 조각 작성
4교시값을 바꾸고 손 추적 연습

먼저 감각으로 이해하기

언어의 옷은 달라도 값의 여행은 같다

`input()`으로 받은 값은 글자다. 공부시간 합계를 계산하려면 `int()`를 통과시켜 숫자로 바꿔야 한다.

C와 Java에서 하던 입력-저장-계산-출력 흐름이 Python에서는 더 짧게 보인다.

input"3"→int3→hours3→출력목표까지 2시간

손가락 실습: 종이에 문자열 `"3"` 카드와 숫자 `3` 카드를 따로 만들어 `int()` 문을 통과시키며 교체한다.

직접 따라치기

첫 프로그램은 하루 공부시간과 목표의 차이를 보여 준다.

코드를 입력한 뒤 반드시 실행한다

goal = 5
hours = int(input("오늘 공부시간: "))

print("오늘 기록:", hours, "시간")
if hours >= goal:
    print("목표 달성!")
else:
    print("목표까지", goal - hours, "시간 남음")

실행 결과 예

오늘 공부시간: 3 오늘 기록: 3 시간 목표까지 2 시간 남음

한 줄씩 읽어 보기

  1. `input()` 결과를 `int()`가 정수로 바꾸어 hours에 넣는다.
  2. 3 >= 5는 거짓이므로 else가 실행된다.
  3. 목표와의 차이 5 - 3을 바로 출력한다.
입력한 뒤 바꾸어 볼 값

입력을 5와 7로 바꾸어 조건 경계와 목표 초과 상황의 출력을 예측한다.

손으로 추적하는 문제

Python도 시험에서는 들여쓰기 범위와 변수값 변화가 핵심이다.

score = 70
if score >= 60:
    score = score + 5
else:
    score = score - 5
print(score)

문제. 무엇이 출력되는가?

정답과 풀이 열기

70은 60 이상이므로 5를 더해 `75`가 출력된다.
