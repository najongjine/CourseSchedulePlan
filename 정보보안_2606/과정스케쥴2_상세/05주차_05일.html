입력한 다섯 날 기록을 분석하여 총합, 평균, 최고, 목표 달성일을 보여 주는 프로그램을 완성한다.

먼저 감각으로 이해하기

데이터가 쌓이면 질문을 던질 수 있다

리스트는 단순한 저장소가 아니라 '총 몇 시간?', '가장 긴 날?', '목표를 몇 번 달성?' 같은 질문에 답할 재료다.

쇼핑몰에서도 상품과 주문 데이터가 쌓인 뒤 똑같이 합계와 조건을 묻게 된다.

입력5일 시간→리스트[...]→분석sum/max/count→리포트결과 출력

손가락 실습: 네 명의 샘플 기록을 받아 칠판에 리스트로 쓰고, 프로그램 결과와 손계산이 같은지 확인한다.

직접 따라치기

필수 코드 전체를 입력하고 내 기록 또는 제공된 테스트 기록으로 실행한다.

코드를 입력한 뒤 반드시 실행한다

def count_success(hours, goal):
    count = 0
    for value in hours:
        if value >= goal:
            count += 1
    return count

records = []
goal = 3
for day in range(5):
    hours = int(input(f"{day + 1}일차 공부시간: "))
    records.append(hours)

total = sum(records)
print("=== 주간 분석 ===")
print("총 공부시간:", total)
print("하루 평균:", total / len(records))
print("최고 기록:", max(records))
print("목표 달성일:", count_success(records, goal))

실행 결과 예

1일차 공부시간: 2 2일차 공부시간: 4 3일차 공부시간: 3 4일차 공부시간: 1 5일차 공부시간: 5 === 주간 분석 === 총 공부시간: 15 하루 평균: 3.0 최고 기록: 5 목표 달성일: 3

한 줄씩 읽어 보기

  1. 입력될 때마다 리스트가 한 칸씩 길어진다.
  2. 분석 결과는 같은 records 데이터를 서로 다른 방식으로 읽은 값이다.
  3. 함수는 records를 바꾸지 않고 조건에 맞는 개수만 반환한다.
입력한 뒤 바꾸어 볼 값

선택 기능으로 공부시간이 가장 낮은 값도 출력하고, 왜 보충 학습 메시지와 연결할 수 있는지 적는다.

손으로 추적하는 문제

리스트가 함수에 전달되는 코드는 함수 내부 반복표를 만들면 안정적으로 풀 수 있다.

def calc(data):
    total = 0
    for x in data:
        if x >= 3:
            total += x
    return total
values = [2, 4, 3, 1]
print(calc(values))

문제. 출력값은 무엇인가?

정답과 풀이 열기

3 이상인 4와 3만 합산한다. 출력은 `7`이다.

감각을 굳히는 연습

먼저 종이에 예상 결과를 적고, 그 다음 코드를 바꾸어 실행해서 확인한다. 틀렸다면 틀린 이유가 학습 기록이다.

  1. 기록 `[2,4,3,1,5]`로 시연하고 손계산 결과와 대조한다.
  2. 목표시간을 4로 바꿔 달성일 결과가 달라짐을 확인한다.
  3. 리스트, 함수, 조건문이 각각 있는 줄을 표시한다.
